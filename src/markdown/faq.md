# 🧩拼图助手 常见问题

## 数据安全

1. 所有的数据均在本地设备存储，不会上传到服务器。
2. 摄像机的使用也仅仅为了获取 拼图的图片信息，不会额外收集其他设备。

## 如何使用?

1. 打开 🧩拼图助手 应用。
2. 访问 "设置" -> "加载示例" -> 再点击右上角，即可预览内置的示例图片以及相关结果


## 名词与解释


### 参考图（背景图）

- 完整的大图

### 匹配图（含有拼图碎片图片）

- 含有 拼图碎片 的小图

### 颜色校准图片, 什么时候需要上传校准图片

- 当 参考图（背景图） 与 匹配图（含有拼图碎片图片） 颜色差异较大时，需要上传校准图片，以提高匹配准确性。
- 校准图片必须是 参考图（背景图）的完整版，可以比 参考图小，但是 **颜色** 要与匹配图 保持一致。
- 如果无法获取到 "完美参考图"，则可以上传校准图片，以提高匹配准确性。

### 什么是“置信度”?

- 置信度是指 应用 对 拼图碎片 匹配结果的 可信度 或 准确性。
- 置信度越高，说明 应用 对 拼图碎片 匹配结果的 准确性 越高。


## 更多设置参数

### 参数: 匹配模式 

1. 精准优先
   - 速度最慢，准确度越高
2. 均衡模式
3. 速度优先
   - 速度最快，但是准确度可能会低一些

> Tips:
>> 省点模式下，计算速度会显著下降。

### 参数: 匹配拼图数量

- 一次最多匹配照片（图片）中的 🧩拼图碎片 的数量
- 置信度靠前的 N 个结果


### 参数: 清除历史记录

- 关闭之后，更换匹配图片（含有拼图碎片图片）时，不会清除之前的数据
- 如果更换背景图（参考图），则会清除之前的数据

### 参数: 调试模式

- 开启之后，会显示更多的匹配过程的信息


## 为什么不准确?

- 步骤 1：是基于 AI模型 识别 "🧩拼图碎片", 
- 步骤 2：再由 "🧩拼图碎片" 使用传统的匹配模式匹配最终参考图

### 纯色系，无法识别

- 匹配算法使用的是传统模型算法，对于大量同色的位置来说，效果不好。例如 大量的 天空、海洋

### 可以更好的处理

- 多颜色，多纹理特征的拼图系列
- 颜色鲜明的漫画、动画系列

## 如何提高识别结果?

1. 平行与拍摄照片, 如果图片倾斜拍摄到的图片会存在 一些失真，可能导致识别效果差
2. 更清晰的参考图
   - 同时也要注意 更大的参考图也对应着更久达到匹配时间
3. 正确的设置网格行列
   - 行列信息会直接关系到 匹配图片的范围准确度，正确的设置行列，可以大幅度提高准确性。
