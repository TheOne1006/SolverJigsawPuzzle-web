# 🧩拼图助手 - 产品展示说明

## 一、产品概述

🧩拼图助手是一款运行于 iOS 设备上的本地拼图辅助工具。通过对用户拍照或从相册选择的拼图图片进行智能识别和分析，帮助快速定位目标拼图块在整体拼图中的大致位置，减轻找块压力，让大拼图也能轻松完成。

本应用为纯离线工具，核心处理流程均在设备本地完成，正常使用过程中无需注册或登录账户。

## 二、核心功能

### 1. 拍照 / 上传拼图图片

支持使用相机拍摄当前拼图桌面，也支持从系统相册中选择已拍摄的拼图照片，灵活适配不同使用习惯。

### 2. 自动识别拼图区域

基于图像识别算法，对图片中的拼图区域进行分析，过滤背景干扰，提高后续定位准确度。

### 3. 定位目标拼图块位置

用户在应用中标记或选择需要查找的拼图块图片后，应用会在整体拼图图像中进行匹配，给出该拼图块大致所属区域，辅助用户快速找到对应位置。


## 三、典型使用场景

- 拼图过程中，某一块始终找不到合适的位置，希望快速缩小范围；
- 想要在短时间内完成拼图，但又不希望完全靠“蒙”，需要一定的智能辅助；
- 亲子拼图、聚会拼图场景中，帮助小朋友或新手更有参与感。

## 四、隐私与安全

- 图片处理和识别过程全部在设备本地完成，不会上传到任何服务器；
- 应用仅在您授权的前提下访问相机和相册权限，用于选取或拍摄拼图图片；
- 正常使用过程中无需登录账户，也不会收集姓名、手机号、邮箱等可识别您身份的信息；
- 具体个人信息处理和权限使用说明，请参见《🧩拼图助手隐私政策》。

## 五、使用须知

- 本应用仅作为拼图爱好者的辅助工具，旨在提升拼图体验与效率；
- 请勿将本应用用于任何违法、违规或违反第三方平台规则的用途；
- 拼图图片涉及他人作品或受版权保护内容时，请在合法、合理的范围内使用本应用。
